# react-native-swiftui-tabview

SwiftUI TabView in React Native

## Installation

```sh
npm install react-native-swiftui-tabview
```

## Usage


```js
import { SwiftuiTabviewView } from "react-native-swiftui-tabview";

// ...

<SwiftuiTabviewView color="tomato" />
```
